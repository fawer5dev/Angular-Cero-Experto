# Angular: De cero a experto

This repository contains my code and exercises from the Udemy course [Angular: De cero a experto](https://www.udemy.com/course/angular-fernando-herrera/) by Fernando Herrera.

## 📚 Course Description

A comprehensive course covering Angular from the basics to advanced topics, including TypeScript fundamentals, component architecture, routing, services, and modern Angular development practices.

## 🗂️ Project Structure

```
├── 01-typescript-intro/     # TypeScript fundamentals
│   ├── src/
│   │   └── topics/          # TypeScript topics and exercises
│   └── index.html
│
└── 02-bases/                # Angular basics
    ├── src/
    │   └── app/             # Angular application components
    └── angular.json
```

## 📖 Sections

### 01 - TypeScript Introduction
Introduction to TypeScript covering:
- Basic types
- Interfaces and objects
- Functions
- Destructuring
- Classes
- Generics
- Decorators
- Optional chaining

### 02 - Angular Bases
Angular fundamentals including:
- Components
- Templates
- Data binding
- Event handling
- Component architecture

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn
- Angular CLI

### Installation

Each section has its own dependencies. Navigate to the specific folder and run:

```bash
cd 01-typescript-intro
npm install
```

or

```bash
cd 02-bases
npm install
```

### Running the Projects

#### TypeScript Section
```bash
cd 01-typescript-intro
npm run dev
```

#### Angular Bases
```bash
cd 02-bases
npm start
```
